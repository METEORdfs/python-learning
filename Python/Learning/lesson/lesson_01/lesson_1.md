## Python 注释
和传统语言的“//”不同，python的注释是“#”。  

从‘#’开始，直到行末，都是注释。

作用： 
1. 帮助阅读代码
2. 忽略代码

```python
print("hello world")  # 这是正常输出

# print("hello world") # 这行被注释了，不会执行
```


多行注释：
```python
"""
这是一个多行字符串，
它被当做多行注释来用，
因为没有被赋值，所以解释器忽略它。
"""
print("这行代码会执行")
```

## Python 变量
变量是存储数据的容器，是关联一个对象的符号（标识符），会在运行时创建，并分配内存。

变量的命名规则：
1. 变量名只能包含字母、数字和下划线

```python
a = 1
a1 = 2
a_1 = 3
a@ = 1 # 错误
```

2. 变量名必须以字母或下划线开头，其后可以跟字母、数字和下划线

```python
_a = 1
a = 2
a1 = 3
a_1 = 4
```

3. 严格区分大小写

```python
a != A
```

4. 不能使用Python关键字作为变量名

```python
if = 1 # 错误
```
## Python 数据类型

数字类型：
1. 整数（int）
2. 浮点数（float）
3. 复数（complex）

整数常量的表示：
1. 10进制：123
2. 16进制（0x开头）：0xFF
3. 8进制（0o开头）：0o77
4. 2进制（0b开头）：0b1010

浮点数常量的表示：
1. 十进制：123.456
2. 科学计数法：1.23e-4  
    格式：小数+e/E+整数，其中e/E表示10的指数，最后的整数表示指数的值  
    e.g. 1.23e-4 = 1.23 * 10^(-4) = 0.000123

复数常量的表示：
1. 带复数符号的数字：123 + 456j
2. 格式：实部 + 虚部j/J

布尔类型：bool
1. 布尔类型有两个值：  
    真：True  
    假：False


空类型：None
1. None表示空对象，不表示任何类型
2. None对象不能进行运算
3. 是唯一一个不可比较的对象

作用：占位符  

说明：空值和任何其他对象比较永远为假

## 运算符（操作符）和 表达式

运算符：完成运算的符号

算数运算符： + - * / % ** //  

1. '+' 加  

```python
a = 10
b = 20
print(a + b) # 30
```

2. '-' 减  
    
```python
a = 10
b = 20
print(a - b) # -10
```

3. '*' 乘

````python
a = 10
b = 20
print(a * b) # 200
````

4. '/' 除

```python
a = 10
b = 20
print(a / b) # 0.5
```

5. '%' 取余数

```python
a = 10
b = 20
print(a % b) # 10
```

6. '**' 幂

```python
a = 10
b = 2
print(a ** b) # 100
```

7. '//' 取整除

```python
a = 10
b = 20
print(a // b) # 0
```

## is & is not 运算符

is 和 is not 是比较两个对象是否相等的运算符
1. is：判断两个对象是否相等  
2. is not：判断两个对象是否不相等

## del 语句
作用：删除对象

语法格式： del [对象]

e.g.
```python
a = 10
del a
print(a) # NameError: name 'a' is not defined
```
## 复合赋值运算符

1. +=  
```python
    a += 10  # 等价于 a = a + 10
    print(a)
```

2. -=  
```python
a -= 10 # 等价于 a = a - 10
print(a)
```

3. *=  a *= 10 等价于 a = a * 10
4. /=  a /= 10 等价于 a = a / 10
5. %=  a %= 10 等价于 a = a % 10

## 关系运算符

1. '=='  判断两个对象是否相等
2. '!='  判断两个对象是否不相等
3. '>'   判断第一个对象是否大于第二个对象
4. '<'   判断第一个对象是否小于第二个对象
5. '>='  判断第一个对象是否大于等于第二个对象
6. '<='  判断第一个对象是否小于等于第二个对象
7. '==='  判断两个对象是否相等
